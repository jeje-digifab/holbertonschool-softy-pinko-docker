### Task 0: Create Your First Docker Image

**Objective**: Create a Docker image that echoes "Hello, World!" when run.

**Description**:
- This task involves creating a Docker image using a Dockerfile.
- The Dockerfile is based on the latest Ubuntu image.
- The image updates APT and upgrades installed software.
- When the Docker container runs, it echoes "Hello, World!" to the terminal.

**Building and Running**:
- The Docker image is built using a Dockerfile and tagged as `softy-pinko:task0`.
- Running the Docker image in a container will display "Hello, World!" in the terminal.

**Repository Information**:
- **GitHub Repository**: [holbertonschool-softy-pinko-docker](https://github.com/holbertonschool-softy-pinko-docker)
- **Directory**: `task0`

This task demonstrates the fundamental steps of building and running a Docker container, showcasing the creation of a simple Docker image that performs a basic action.
